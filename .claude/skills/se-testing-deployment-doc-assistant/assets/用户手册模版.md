## 格式规范说明

| 元素     | 格式             | 用途               |
| -------- | ---------------- | ------------------ |
| 章节标题 | `##` / `###`     | 层级结构，最多四级 |
| 提示信息 | `> **xxx**: `    | 版本、日期等元信息 |
| 表格     | `\| 表头 \|`     | 参数对照、版本对比 |
| 代码块   | ` ```bash/yaml ` | 命令、配置示例     |
| 步骤列表 | `1. 2. 3.`       | 操作流程           |
| 无序列表 | `-` / `*`        | 功能点列举         |
| 强调     | `**粗体**`       | 关键术语、按钮名   |
| 引用     | `>`              | 注意事项、警告     |
