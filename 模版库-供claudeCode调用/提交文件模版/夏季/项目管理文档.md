# 项目管理文档模版

## 文档说明

- **用途**：记录软件项目全生命周期的管理过程，包括迭代进度、工具使用、版本控制及质量保障。

- **适用阶段**：需求分析、设计、开发、测试、部署及维护阶段。

- 填写规范

  ：

  - 所有 `[ ]` 标记处需替换为实际项目信息。
  - 图片引用请使用相对路径或在线链接，并附带简要说明。
  - 日期格式统一为 `YYYY-MM-DD` 或 `第 X 天`。
  - 遇到风险或阻碍时，必须记录“原因分析”与“应对措施”。
