## 格式规范总览

| 元素 | 语法 | 使用场景 |
|-----|------|---------|
| 文档元信息 | `> **字段**: 值` | 版本、日期、状态 |
| 层级标题 | `#` `##` `###` `####` | 最多四级，严格层级 |
| 表格 | `\| 列1 \| 列2 \|` | 参数、对比、清单 |
| 代码块 | ` ```语言 ` | 命令、配置、代码 |
| 步骤列表 | `1. 2. 3.` | 操作流程 |
| 无序列表 | `-` 或 `*` | 功能点、注意事项 |
| 警示标识 | `⚠️` `💡` `📌` | 警告、技巧、注意 |
| 界面元素 | `【元素名】` | 按钮、菜单、路径 |
| 输入内容 | `` `内容` `` | 需输入的文字 |
| 状态复选 | `[ ]` `[x]` | 检查清单 |
| 引用块 | `>` | 补充说明、引用 |
